Process to run the app:

USING ZIP FILE:
1. extract the zip file.
2. open the folder in vs-code.
3. open terminal and run the command `npm install`.
4. after installing all the dependencies run the command `npm start`.
5. open the browser and go to `http://localhost:3000/` to see the app running.

USING GIT REPOSITORY:
1. Open terminal and clone the repository. Use this command:  git clone https://github.com/hello-snsa/directory.git
2. open the folder in vs-code.
3. open terminal and run the command `npm install`.
4. after installing all the dependencies run the command `npm start`.
5. open the browser and go to `http://localhost:3000/` to see the app running.
